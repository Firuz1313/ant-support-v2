Техническое описание для владельца
=================================

Как работает сайт:
------------------
1. Пользователь заходит на сайт и видит виртуальный пульт и экран ТВ.
2. Нажимая на кнопки пульта, пользователь отправляет команды (например, смена канала, громкость, открытие меню и т.д.).
3. Экран ТВ реагирует на команды: меняется канал, открывается список каналов, появляется информация о выбранном канале и т.д.
4. Всё взаимодействие происходит мгновенно — сайт работает как Single Page Application (SPA).

Технологии, фреймворки и библиотеки:
------------------------------------
- React — основной фреймворк для построения интерфейса.
- TypeScript — строгая типизация для надежности и удобства разработки.
- React Context API — для глобального управления состоянием (например, состояние ТВ, выбранный канал и т.д.).
- React Hooks — для управления состоянием и жизненным циклом компонентов.
- react-icons — для красивых иконок на кнопках и в интерфейсе.
- CSS-in-JS (inline-стили) — для стилизации компонентов (градиенты, тени, анимации).
- Tailwind CSS (судя по наличию tailwind.config.ts) — возможно, используется для утилитарных классов и быстрой стилизации.
- Vite — современный сборщик для быстрого старта и разработки.
- PostCSS — для обработки CSS.
- Netlify — для деплоя (есть netlify.toml).

Структура проекта:
------------------
- src/components/ — основные компоненты (пульт, экран, layout и т.д.).
- src/context/ — контексты для управления состоянием.
- src/pages/ — страницы приложения.
- src/hooks/ — кастомные хуки.
- src/lib/ — вспомогательные функции.
- public/ — статические файлы (иконки, favicon и т.д.).
